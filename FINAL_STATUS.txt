OPTION 1: Install PostgreSQL (Recommended)
-------------------------------------------
1. Download: https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
2. Install PostgreSQL 16 (password: postgres)
3. Run: INSTALL_AND_RUN.bat
   OR manually:
   - createdb -U postgres dpis_db
   - psql -U postgres -d dpis_db -f backend\config\init.sql
4. Restart backend server

OPTION 2: Use Online Database
------------------------------
1. Sign up for free PostgreSQL: https://www.elephantsql.com/
2. Get connection string
3. Update backend\.env with connection details
4. Restart backend server

OPTION 3: Use Docker (if installed)
------------------------------------
1. Install Docker Desktop
2. Run: docker-compose up -d postgres mongodb
3. Initialize: docker-compose exec postgres psql -U postgres -d dpis_db -f /docker-entrypoint-initdb.d/init.sql
4. Start backend and frontend
